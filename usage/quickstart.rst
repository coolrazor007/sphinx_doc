Quickstart Setup
================


"cd" to destination for the doc

.. code-block:: bash
  :linenos:

  sphinx-quickstart



choose mostly defaults (for now, not separating build and source directories)


Render the Doc
================

make [builder]  <--e.g. html latexpdf
